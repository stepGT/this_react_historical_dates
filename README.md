# Исторические даты

Интерактивный блок с временными отрезками и событиями.

## Технологии

- TypeScript
- React.js
- SCSS
- Webpack
- Swiper (слайдеры)
- GSAP (анимации)

## Функциональность

- Отображение временных отрезков (от 2 до 6)
- Интерактивная навигация по кругу
- Анимированное изменение дат при переключении
- Слайдер с событиями для активного периода
- Адаптивная верстка
- Независимые блоки (можно добавить несколько на страницу)

## Установка и запуск

```bash
git clone https://github.com/stepGT/this_react_historical_dates
cd this_react_historical_dates
npm install
npm start

```
